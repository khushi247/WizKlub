# WizKlub Lead Generation Chatbot

AI-powered lead generation chatbot built with Streamlit + Groq (Llama 3.3 70B).

## Setup — Streamlit Cloud Secrets

Go to your app → ⋮ → Settings → Secrets and add:

```toml
GROQ_API_KEY = "gsk_your_key_here"
```

Get a free key at **console.groq.com** → API Keys → Create API Key
